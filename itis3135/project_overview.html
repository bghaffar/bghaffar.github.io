Project Overview: Fitness Philosophy

This website is being developed to support Faridun Jabborov’s fitness journey and serve as an online platform where he can eventually onboard clients for personal training or coaching services. The purpose of this site is to document and share his fitness philosophy, routines, and useful content for anyone interested in health and exercise. Over time, it will grow into a potential base for a fitness coaching business.

Intended Users

Users are people interested in fitness—beginners to intermediate—who want easy access to structured workouts and inspiration.

Content Overview

  • Home Page: Introduction to the site
  • Workout Plans: Structured routines by day
  • Gallery: Visual display of workout equipment and techniques
  • Contact Page: To reach out for consultations
  • User Form: To gather personalized preferences

Page Design

  • home.html: Welcome page - everyone - general info - no form - links to other pages.
  • plans.html: Workout plans - customers - muscle-focused routines - no form - includes filter buttons.
  • gallery.html: Exercise visuals - all users - interactive accordion gallery using jQuery UI.
  • contact.html: Consultation request - users - form with validation and AJAX feedback.
  • userform.html: Form to submit preferences - users - validated form inputs, preview on submit.

Dynamic Functionality

- Plans page includes filterable sections using jQuery
- Gallery uses jQuery UI accordion widget
- Contact form submits via AJAX and shows a response
- User form includes JS validation and preview output
